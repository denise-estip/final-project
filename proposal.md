# Todays Outfit!

## Repository
https://github.com/denise-estip/final-project.git

## Description
What I plan to make is a simple dress up game where there will be a base model and clothing sprites you can scroll through with arrow buttons. This is relevant to media and digtal arts because I will be combining my art (the sprite images) and coding through an interactive game. 

## Features
- Feature 1
	- Clicking the arrow cycles through clothing options possibly using an object class and pygame.mouse 
- Feature 2
	- A bobbing text animation at the top that says "Todays Outfit!", similar to how we did twikling animation with transparency in animated rain module. 
- Feature 3 
	- arrow sprite changes if mouse is hovered over it using pygame.mouse
- Feature 4
    - Adding sound effects with each button press

## Challenges
- How I could implement a functional button that registers when it clicks.
- The abilty to blit images in layers and not have them awkwardly overlap.
- Using the button clicks to select different clothes. 
- Adding sound effects to each button press 

## Outcomes
Ideal Outcome:
- The start screen is a character and arrows, with animated text floating at the top with game title. 
- Each button works properly and blits a clothing item when clicked
- audio plays when button is clicked

Minimal Viable Outcome:
- Everything is blit on screen properly
- button audio is heard when clicked
- button click shows another clothing item on screen
