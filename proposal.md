## Description
What I plan to make is a simple dress up game where there will be a base model and clothing sprites you can scroll through with arrow buttons. This is relevant to media and digtal arts because I will be combining my art (the sprite images) and coding through an interactive game. 

## Features
- Feature 1
	- Clicking the arrow cycles through clothing options possibly using an object class and pygame.mouse 
- Feature 2
	- A bobbing text animation at the top that says "Todays Outfit!", similar to how we did twikling animation with transparency in animated rain module. 
- Feature 3 
	- arrow sprite changes if mouse is hovered over it using pygame.mouse
- Feature 4
    - Adding sound effects with each button press
